# User Management Scripts

This repository features two shell scripts designed to manage user accounts on a Linux system:

_______________

useradd.sh

This script automates the creation of multiple user accounts in a loop. Like user1, user2, user3 until the loop ends. The password input is optional and can be skipped if not needed.

_______________

userdel.sh

This script automates the deletion of multiple user accounts in a loop. It removes users like user1, user2, user3 until the loop ends.
